# 🎬 BotFlix

> Seu assistente pessoal inteligente para criar o filme perfeito 🍿✨

O **BotFlix** é uma aplicação desenvolvida para ajudar usuários a descobrirem filmes ideais com auxílio de Inteligência Artificial, automações inteligentes e integração com APIs de cinema.

A proposta do projeto é unir experiência personalizada, recomendações inteligentes e automação para transformar a escolha de filmes em algo rápido, divertido e eficiente.

---

# 🚀 Tecnologias Utilizadas

| Tecnologia | Descrição |
|------------|------------|
| 🌐 HTML5 | Estrutura da aplicação |
| 🎨 CSS3 | Estilização e responsividade |
| ⚡ JavaScript | Funcionalidades e interações |
| 🤖 N8N | Automações inteligentes |
| 🧠 Gemini IA | Integração com Inteligência Artificial |
| 🎥 TMDB API | Base de dados de filmes |

---

# 🧠 Funcionalidades

- ✅ Recomendações inteligentes de filmes  
- ✅ Integração com IA Gemini  
- ✅ Consulta de filmes via API TMDB  
- ✅ Interface moderna e responsiva  
- ✅ Automações utilizando N8N  
- ✅ Experiência personalizada para o usuário  

---

# 🎯 Objetivo do Projeto

O objetivo do **BotFlix** é oferecer uma experiência inteligente para recomendação de filmes, utilizando IA para entender preferências do usuário e sugerir opções alinhadas ao seu perfil e momento.
